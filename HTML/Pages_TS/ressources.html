L’accès à Apache consiste à atteindre le serveur web via une adresse IP ou un nom de domaine, généralement par le protocole HTTP (port 80) ou HTTPS (port 443). Il se fait depuis un navigateur, un client HTTP ou un autre service réseau. L’accès peut être contrôlé par la configuration d’Apache (fichiers httpd.conf ou .htaccess), qui définit les autorisations, restrictions, répertoires accessibles et règles de sécurité.
